# WeekFlow Project Status

## Project
WeekFlow

## Summary

WeekFlow is a task and goal planning app focused on weekly productivity.

The app is built around the idea that the user should be able to quickly capture tasks, plan the week, complete tasks, and look back at what they finished.

The app also includes 12-week goals so smaller tasks can connect to larger goals.

---

## Current Main Screens

WeekFlow currently has these tabs:

```txt
Goals
Inbox
Daily
Weekly
History
```

---

## Current Features

The app currently supports:

```txt
Creating tasks
Creating goals
Saving tasks locally with SQLite
Saving goals locally with SQLite
Adding tasks to Inbox
Moving Inbox tasks into weekdays
Viewing tasks by day
Viewing the full week
Completing tasks
Viewing completed tasks in History
Deleting tasks
Deleting goals
Adding task notes
Adding task priority
Showing notes and priority across task screens
Linking tasks to goals
Showing linked tasks under goals
Showing linked goals in Daily, Weekly, and History
```

---

React Native

## Current Status

The app is now a working local MVP.

It has enough functionality to demonstrate requirements, design, local data persistence, reusable architecture, navigation, state management, and a complete task workflow.
