# WeekFlow Roadmap

## Project
WeekFlow

## Purpose

This roadmap lists the major features planned for WeekFlow.

The goal is to build the app in small working checkpoints instead of trying to add everything at once.

---

## Completed

```txt
Create Expo app
Set up tab navigation
Add Goals tab
Add Daily tab
Add Weekly tab
Add History tab
Add Inbox tab
Add SQLite persistence for tasks
Add SQLite persistence for goals
Add task delete controls
Add goal delete controls
Add task notes
Add task priority
Show notes and priority across task screens
Link tasks to 12-week goals
Show linked tasks under goals
Show linked goals in Daily
Show linked goals in Weekly
Show linked goals in History
```

## Long-Term Goal

The long-term goal is for WeekFlow to become a clean weekly planning app that works across devices and can optionally sync with a self-hosted backend.
