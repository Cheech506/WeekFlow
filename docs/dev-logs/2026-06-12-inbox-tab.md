# Dev Log — Added Inbox Tab

## Date
June 12, 2026

## Project
WeekFlow

## Summary

Today I added the Inbox tab.

The reason I added this was because I wanted a place to quickly write down tasks without having to decide what day they belong on right away.

Before this update, tasks had to be added directly to Daily or Weekly. That worked, but it was not ideal for quick ideas or random tasks that come up during the day.

After this update, I can add a task to Inbox first, then move it into the week later.

---

## What I Added

I added a new Inbox tab to the app.

The Inbox lets the user:

```txt
Add an unscheduled task
View unscheduled tasks
Move a task to Monday through Sunday
Mark a task complete
Delete a task
```

---

## Why I Added This

The Inbox is useful because not every task has a planned day right away.

Sometimes I just need to write something down quickly so I do not forget it.

The Inbox works like a temporary holding area before the task becomes part of the weekly plan.

---

## How It Works

```txt
User adds task to Inbox
→ Task is saved with day = "Inbox"
→ Inbox shows the task
→ User chooses a weekday
→ Task day updates in SQLite
→ Task disappears from Inbox
→ Task appears in Weekly under that day
```
